# CapyChop

A fun and interactive capybara-themed image chopping game where players try to slice an image to match a target percentage.

## Features

- Random goal percentages for each round
- Animated knife that follows your cursor
- Star rating system based on accuracy
- Background music and sound effects
- Track your performance over the last 5 rounds
- Settings panel with music toggle

## How to Play

1. Click "Start Game" to begin
2. A random goal percentage will be shown
3. Move your mouse/finger across the image
4. Click to chop the image at your desired position
5. Try to match the goal percentage as closely as possible
6. Earn up to 10 stars based on your accuracy!

## Technologies Used

- Pure HTML5, CSS3, and JavaScript
- Canvas API for image manipulation
- Web Audio API for sound effects

## Local Development

Simply open `index.html` in a modern web browser.
